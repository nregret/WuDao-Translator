# 优化替换规则模块 - 删除规则管理按钮

## 📋 优化目标
删除"保存规则"、"加载规则"、"清空规则"三个按钮，只保留"添加规则"和删除单个规则的功能。

## 🎯 实现步骤

### 1. 修改 BatchTextReplace.vue 的规则区域

**修改内容**：
- 将 `textarea` 改为 `input type="text"`，单行输入
- 规则布局改为横向：左侧"被替换"输入框，右侧"替换为"输入框
- 减小规则项的 padding 和间距
- 删除规则管理按钮区域（保存、加载、清空）
- 删除相关的函数：`saveRules`、`loadRules`、`clearRules`
- 删除 `onMounted` 中的 `loadRules()` 调用

**新的规则区域结构**：
```vue
<!-- 替换规则区域 - 支持多条规则 -->
<div class="replace-rules-section">
  <div class="rules-header">
    <span class="rules-title">替换规则</span>
    <button @click="addRule" class="add-rule-btn">
      <span class="btn-icon">➕</span>
      <span class="btn-text">添加规则</span>
    </button>
  </div>
  
  <div class="rules-list">
    <div 
      v-for="(rule, index) in rules" 
      :key="index" 
      class="rule-item"
    >
      <div class="rule-number">规则 {{ index + 1 }}</div>
      <div class="rule-inputs">
        <div class="input-group">
          <label>被替换</label>
          <input 
            v-model="rule.searchText" 
            placeholder="输入要查找的文本..."
            class="rule-input"
          />
        </div>
        <div class="input-group">
          <label>替换为</label>
          <input 
            v-model="rule.replaceText" 
            placeholder="输入替换后的文本..."
            class="rule-input"
          />
        </div>
      </div>
      <button @click="removeRule(index)" class="remove-rule-btn" v-if="rules.length > 1">
        🗑️
      </button>
    </div>
  </div>
</div>
```

### 2. 更新样式

**样式优化**：
- 减小 `.rule-item` 的 padding：从 `16px` 改为 `10px`
- 减小 `.rule-inputs` 的 gap：从 `12px` 改为 `8px`
- `.rule-input` 改为单行输入，高度约 `36px`
- `.rule-number` 更紧凑：padding 减小，字体稍小
- 删除按钮调整：位置和大小更合适
- 删除 `.rule-actions` 相关样式
- 整体规则列表间距减小

### 3. 清理代码

**删除的函数**：
- `saveRules()` - 保存规则到 localStorage
- `loadRules()` - 从 localStorage 加载规则
- `clearRules()` - 清空规则

**删除的导入**：
- 无需删除，只删除函数定义

## 📁 修改文件
1. **修改**: `src/components/BatchTextReplace.vue`
   - 模板部分：删除规则管理按钮区域，优化规则布局
   - 脚本部分：删除 saveRules、loadRules、clearRules 函数，删除 onMounted 中的 loadRules 调用
   - 样式部分：删除 rule-actions 相关样式，优化规则项样式
