# 批量文本替换页面 UI/UX 优化实施计划

## 📋 优化目标
将 [BatchTextReplace.vue](e:\Project\VsCode\Translater\SimpleTauri\python-in-tauri\src\components\BatchTextReplace.vue) 中的 emoji 图标替换为专业的 SVG 图标，并优化整体 UI/UX 体验。

## 🎯 具体实施步骤

### 步骤 1：替换所有 emoji 图标为 SVG 图标
- **标题图标**：🔄 → ArrowsRightLeftIcon（替换图标）
- **添加规则按钮**：➕ → PlusIcon（加号）
- **删除规则按钮**：🗑️ → TrashIcon（删除）
- **选择文件按钮**：📄 → DocumentIcon（文件）
- **选择文件夹按钮**：📁 → FolderIcon（文件夹）
- **开始替换按钮**：🚀 → RocketLaunchIcon（火箭）
- **加载中图标**：⏳ → ClockIcon（时钟）
- **保存路径按钮**：📂 → FolderOpenIcon（打开文件夹）
- **移除文件按钮**：× → XMarkIcon（关闭）

### 步骤 2：优化图标样式
- 统一图标尺寸：width="20" height="20"
- 统一 viewBox="0 0 24 24"
- 添加合适的颜色和过渡效果
- 确保图标在 hover 状态下有视觉反馈

### 步骤 3：优化交互反馈
- 为所有按钮添加 `cursor: pointer`
- 增强 hover 效果：颜色变化 + 轻微阴影
- 优化 focus 状态：outline + 轻微阴影
- 添加平滑过渡效果

### 步骤 4：优化样式细节
- 调整按钮间距和布局
- 优化输入框 focus 效果
- 改进禁用状态的视觉反馈
- 确保颜色对比度符合可访问性标准

### 步骤 5：验证响应式布局
- 确保移动端（< 768px）布局正常
- 确保平板和桌面端布局正常
- 测试图标在不同尺寸下的显示效果

- 专业、现代的视觉设计
- 一致的 SVG 图标系统
- 流畅的交互体验
- 优秀的可访问性
